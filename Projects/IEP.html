IEP - Electronic Level

October 2021 - March 2022 - Coursework

Objective

Design an electronic level with visual indicator from basic electronic components:

  • 5V power supply
  • MEMs accelerometer
  • Op-Amp
  • Comparator
  • LEDs

Comments

I found this project to be one of the most difficult I have tackled since coming to Cambridge. I didn't especially enjoy electrical physics/engineering at school and this was my first time constructing a non-trivial circuit and my first time using quite sophisticated components.

There were a number of challenges I faced in the course of this project. One somewhat unexpected challenge was the need to be efficient with the use of space on the breadboard. There were many constituent circuits which cascaded to produce the level. Each of these had to be well organised in order to prevent overcrowding of components and connecting cables.
Another challenge I faced was using documentation in order to use unfamiliar components without issue.
